迭代器模式指提供一种方法顺序访问一个聚合对象中的各个元素，而又不需要暴露该对象的内部表示。
### 实现一个迭代器

### 内部迭代器和外部迭代器


1. 内部迭代器

```
var each = function(ary, callback) {
    for (var i = 0, l = ary.length; i < l; i++) {
        callback.call(ary[i], i, ary[i])  // 把下标和元素当作参数传给callback元素
    }
}

each([1, 2, 3], function(i, n) {
    alert([i, n])
})
```
